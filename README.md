# 🦾 MuJoCo Pick-and-Place Lab (Week 1)

This repository contains the Week 1 assignment for the **Medical Robotics & Simulation Lab (MRSL)**.

## 🎯 Objective
Understand and evaluate a pick-and-place task using the Franka Panda robot in MuJoCo.

---

## 📂 Repository Structure

- `starter_code/` → Base MuJoCo simulation files  
- `assignment/` → Assignment description (PDF)  
- `results/` → Student experiment results  
- `docs/` → Reports and documentation  

---

## 🧪 Tasks

Students are required to:

- Understand the system (MuJoCo + Panda robot)
- Modify object position, target, and speed
- Run experiments
- Analyze success/failure

---

## 📊 Expected Output

Each student must submit:

- Modified code  
- Results (CSV or table)  
- Report (1–2 pages)

---

## ⚠️ Rules

- Simulation must be executed (no theoretical answers only)
- Code must be original
- Clear observations required
